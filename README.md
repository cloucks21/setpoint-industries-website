# Setpoint Industries

Brochure site for Setpoint Industries, a commercial refrigeration services company.

Static HTML, no build step. Hosted on Netlify.

## Pages

- `index.html` — Home
- `about.html` — About, advisory team, engagement stages
- `contact.html` — Contact form

## Brand

- Typography: IBM Plex Serif (headings), IBM Plex Sans (body), IBM Plex Mono (eyebrow labels)
- Palette: steel blue (`#1e3a4f`) + copper (`#b87333`) on warm cream (`#f4f1eb`)
- Logo: thermostat-dial mark

## Development

Open any HTML file directly in a browser. No build, no dependencies.
